Что такое UIkit для frontend разработчика?

Скачать CSS и JavaScript можно на официальном сайте getuikit.com.

Подключение классическое. Стили в конец тега , скрипты в конец документа. (Другие варианты подключения, например через Npm можно ознакомиться на github)

Стили:

<link rel="stylesheet" href="css/uikit.min.css" />

Скрипты:

<script src="js/uikit.min.js"></script>
<script src="js/uikit-icons.min.js"></script>

Пример аккордеона

Для определения аккордеона устанавливаем атрибут uk-accordion.

Так же отмечаем заголовки и контентную часть классами class="uk-accordion-title" и class="uk-accordion-content"

Бонусом можно указать класс class="uk-open" определяющий открытый "по умолчанию" пункт.

Множественное открытие

Для множественное открытия нужно устанавливать атрибут со свойством uk-accordion="multiple: true"

Можно комбинировать множественное открытие и запрет закрытие всех так - uk-accordion="multiple: true; collapsible: false"
